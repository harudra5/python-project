# python-project
🏥 Hospital Management System (Python OOP Project)

This project is a Hospital Management System built using Python and Object-Oriented Programming (OOP) concepts. The system allows users to manage patients, doctors, appointments, and billing while storing data in a JSON file for persistence.

The project demonstrates the use of core Python concepts such as classes, inheritance, encapsulation, composition, file handling, exception handling, and data validation.

📌 Features

Add and view patients

Add and view doctors

Book appointments between patients and doctors

Generate hospital bills

View all appointments and billing records

Data stored using JSON file handling

Input validation for age and date

Menu-driven command line interface

🧠 OOP Concepts Used

Classes and Objects – Person, Patient, Doctor, Appointment, Bill, Hospital

Inheritance – Patient and Doctor inherit from Person

Encapsulation – Private attribute for patient ID

Composition – Appointment class connects patient and doctor

Method Usage – Data conversion methods (to_dict())
